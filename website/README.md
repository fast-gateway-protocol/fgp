# FGP Website

Marketing site for the Fast Gateway Protocol project. Built with **React**, **Vite**, **TanStack Router**, and **Tailwind CSS**.

## Project layout

```
website/
├── src/           # React routes + components
├── public/        # Static assets
└── vite.config.ts # Vite configuration
```

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Lint

```bash
pnpm lint
```
